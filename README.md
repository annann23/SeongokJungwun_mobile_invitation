# 🌟 모바일 청첩장 - Next.js + TypeScript + Tailwind

천문우주학과 커플 다운 별빛 테마의 모바일 청첩장을 Next.js, TypeScript, Tailwind CSS로 구현한 프로젝트입니다.


## 🚀 기술 스택

- **Next.js 15**
- **TypeScript**
- **Tailwind CSS**
- **React Hooks**

## 📦 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# 프로덕션 서버 실행
npm start
```
